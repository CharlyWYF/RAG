# 查询改写实验总对照表

## 指标说明

- **命中率**：目标文档命中率，表示每道题的检索来源中是否包含预期目标文档。
- **平均唯一来源数**：每道题最终回答涉及的去重来源文档数的平均值，用于反映来源覆盖范围。
- **平均总耗时(s)**：从问题输入到完整回答输出的平均总时间。
- **平均首字时间(s)**：从问题输入到首次输出回答内容的平均时间。
- **平均改写耗时(s)**：查询改写阶段的平均耗时；无改写组该值为 0。
- **平均检索耗时(s)**：向量检索阶段的平均耗时。
- **平均生成耗时(s)**：最终回答生成阶段的平均耗时。
- **保守回答成功率**：在应保守回答的题目中，系统成功进行保守回答的比例。
- **正确性**：人工评分中回答事实是否正确，0~2 分。
- **完整性**：人工评分中回答是否覆盖关键要点，0~2 分。
- **忠实性**：人工评分中回答是否严格基于检索上下文，0~2 分。
- **检索相关性**：人工评分中检索来源与问题的相关程度，0~2 分。
